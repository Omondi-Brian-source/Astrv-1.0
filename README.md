# Astr Monorepo

This repository contains the Astr SaaS platform, including a Chrome MV3 extension and a Next.js app.

## Packages
- `apps/web` — Next.js 16.1 web app + API routes
- `extension` — Chrome MV3 extension
- `packages/shared` — Shared TypeScript utilities
- `supabase` — Database migrations and seed data
- `scripts` — Local tooling and CLI helpers
- `docs` — Additional documentation

> Initial scaffold only. Implementation will be added in subsequent steps.
